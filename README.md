# Antigravity UI & Design Skills

A curated collection of high-performance AI skills for design, UI development, and architectural decision-making. These skills are optimized for use with the [Antigravity](https://antigravity.google) AI coding assistant.

## Available Skills

- **UX Patterns (`ux-patterns`):** Patterns for modern interface design (auth, pricing, dashboards).
- **UI Design (`ui-design`):** Visual design principles and implementation guides.
- **Accessibility Patterns (`accessibility-patterns`):** WCAG 2.1 compliance and ARIA patterns.
- **Animation Systems (`animation-systems`):** Framer Motion and GSAP orchestration.
- **Vibe Testing (`vibe-testing`):** Visual regression and premium UI heuristics.
- **API Designer (`api-designer`):** Best practices for REST API design.
- **API Standards (`api-standards`):** Security, logging, and middleware standards.
- **Browser Extraction (`browser-extraction`):** Efficient UI extraction parsing and performance rules.
- **Component Library (`component-library`):** Design system architecture and component patterns.
- **Design Tokens (`design-tokens`):** W3C-compatible design token systems.
- **Database Architect (`database-architect`):** SQLite (FTS5) and PostgreSQL (GIN/JSONB) architecture.
- **Monorepo Management (`monorepo-management`):** Turborepo and Nx workspace patterns.
- **Security Hardening (`security-hardening`):** OWASP mitigation and secure session handling.
- **CLI Builder (`cli-builder`):** Architecting robust developer tools.
- **MCP Server Builder (`mcp-server-builder`):** Building Model Context Protocol servers.
- **Docker Playwright (`docker-playwright`):** Containerization for Playwright-based browsers.
- **Skill Forge (`antigravity-skill-forge`):** The meta-skill for building other skills.

## How to Use

1. Clone this repository into your workspace's `.agents/skills` directory.
2. The Antigravity agent will automatically index these skills and use them as context for your requests.
